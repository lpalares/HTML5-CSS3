Material de Estudo

Posso usar qualquer imagem no meu site?

Não é correto pegar uma imagem na internet e usá-las como se fosse sua. Criar um site usando uma imagem que não é sua e publicar na internet pode lhe dar problemas jurídicos, afinal é ilegal

Não posso usar nenhuma imagem?

Algumas imagens são livres para uso legalmente e você pode usá-las a vontade (para uso pessoal e comercial). Essas imagens possuiem a Creative Commons.

Serei punido se usar imagens que não são minhas?

Sim, exceto se a imagem estiver livre para uso comercial ou se você usar para seu projeto/trabalho escolar, por exemplo.
