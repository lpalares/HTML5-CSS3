Formatos de Imagens para Web

Atualmente existem dezenas de formatos de imagens, mas os formatos mais comuns na internet são o png, o jpg, gif e svg. Sendo que o png e o jpg são os que mais se destacam.

PNG ou JPG?

Uma dúvida muito comum é se devemos usar o png ou o jpg no site e a resposta é que devemos usar os dois, no entanto, cada formato tem a hora certa de ser usada. O png possui uma qualidade melhor e permite salvar imagens com fundo transparente, mas é mais pesada que o jpg, o que faz muita diferença na hora de criar um site.

Um site com muitas imagens pesadas fica lento e isso faz a experiência do cliente ser ruim e o Google ou outro navegador vai recomendar menos o site. Por isso é recomendado utilizar o formato jpg, exceto se a imagem precisar de fundo transparente ou se ela precisar de qualidade.

Imagens em celulares

Celulares, por serem menos potentes que computadores, ficam lentos com sites com muitas imagens (principalmente se não forem otimizadas). Você pode resolver isso com tecnologias como media queries e a tag picture que permite mudar a imagem por uma versão mais leve.
